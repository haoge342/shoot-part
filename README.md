This is a project run on WSL2 machine.

## Demo
![Demo](./demo/shoot-part-demo-01.gif)

## Pre-requisite
1. Install WSL2
2. Install CMake
3. Install g++ compiler
4. Install SMFL library

## How to run the project
1. Clone the project
2. Run the following command to run the project
```bash
cd build
./shoot-part
```
